# Task Manager App with TypeScript

A task management web application built with React and TypeScript. The app allows users to create, organize, update, and delete tasks through a clean and responsive interface.

This project focuses on practicing TypeScript fundamentals in a real-world React application while improving component structure, state management, form handling, and user interaction.

## Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Organize tasks by status
- Responsive user interface
- Type-safe components and state
- Reusable React components

## Technologies Used

- React
- TypeScript
- Vite
- CSS
- React Router
- Context API / Redux Toolkit
- Local Storage
